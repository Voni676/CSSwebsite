CSS Lists

Today, we are going to be learning about Lists in CSS

  • list-style type
  • list-style image
  • list-style position

list-style type

In CSS, you can change or alter the way the list is presented (ex: the bullet points in the unordered list)

examples of how the code would be written

  • list-style-type: upper-roman
  • list-style-type: lower-greek
  • list-style-type: lower-alpha
  • list-style-type: simp-chinese-formal

list-style image

This style can change the bullet point or list item to an image.

  • list-style-image:url(example.png)

list-style position

    There are Two Main Types of List-Style Positions
  • list-style position:outisde; the bullet point will be on the outside of the highlighted list item
  • list-style position:inside; the bullet point will be on the inside of the highlighted list item, it will be apart of the text
